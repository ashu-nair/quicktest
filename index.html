⬆ Upload Model Package

Package your trained model as a ZIP containing model.pkl and model_config.json, then upload below.

Drop your ZIP here
or click to browse · model.pkl + model_config.json required
📋 model_config.json Example
{ "model_name": "Iris Classifier", "features": ["sepal_length", "sepal_width", "petal_length", "petal_width"], "target_classes": ["setosa", "versicolor", "virginica"] }
🐍 Create model.pkl Quickly
from sklearn.datasets import load_iris from sklearn.ensemble import RandomForestClassifier import pickle, json, zipfile, os iris = load_iris() clf = RandomForestClassifier().fit(iris.data, iris.target) with open("model.pkl", "wb") as f: pickle.dump(clf, f) with open("model_config.json", "w") as f: json.dump({ "model_name": "Iris Classifier", "features": list(iris.feature_names), "target_classes": list(iris.target_names) }, f) with zipfile.ZipFile("iris_model.zip", "w") as z: z.write("model.pkl") z.write("model_config.json")
